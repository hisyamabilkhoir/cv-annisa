# Annisa Esce Portfolio (HTML/CSS/JS)

Offline-friendly portfolio template for a Content Creator:
- Dark/Light mode (saved in localStorage)
- Smooth section background (no "nabrak" seam)
- 3D tilt cards
- Projects filter + modal
- Floating quick buttons (WA/IG/TikTok + Theme)

## Run
Just open `index.html` in your browser.

## Customize
Edit links in `script.js`:
- instagram, tiktok, youtube, email, whatsapp

Replace placeholder hero photo in `index.html` (data-uri) with your own image path.
